##  Key Highlights
- Built a scalable recommendation engine using machine learning
- Optimized similarity computation for fast recommendations
- Clean and interactive UI using Streamlit
- Real-time movie poster fetching using TMDB API
##  Dataset Information
- Dataset contains movie metadata such as:
  - Title
  - Genres
  - Keywords
  - Cast
  - Crew
- Data preprocessing includes:
  - Handling missing values
  - Feature extraction
  - Text vectorization
##  Model Details
- Algorithm: Content-Based Filtering
- Vectorization: CountVectorizer
- Similarity Metric: Cosine Similarity
- Output: Top N similar movies
##  Performance
- Fast recommendation generation
- Works efficiently even with large datasets
- Lightweight and easy to deploy
